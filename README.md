# ScienceQtech – Employee Performance Mapping (MySQL)

## Project Summary
SQL-based analysis project using **MySQL** to evaluate employee performance and generate business insights such as departmental performance, ranking of employees, and KPI-style summaries.

## Tech Stack
- MySQL

## Key Work
- Used SQL to analyze employee performance data with **JOINs, GROUP BY, ORDER BY, subqueries, and conditional logic**
- Created department-wise summaries and employee-level comparisons
- Produced outputs that support performance tracking and decision-making

## Repository Structure
- `sql/` → SQL scripts (schema + analysis queries)
- `screenshots/` (optional) → query output screenshots

## How to Run
1. Open MySQL Workbench (or any MySQL client).
2. Run:
   - `01_schema.sql` (if included)
   - `02_queries.sql`
3. Execute queries and review results in the output console/grid.

## Notes
If the original dataset is not included due to privacy/licensing, the SQL scripts are still provided to demonstrate the workflow and logic.
